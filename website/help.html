How to Use:

Sending a code:
  • Start Extension in Chrome.
  • Enter a Blackboard Collaborate.
  • Type this string: //code (Don't press Enter or Send).
  • Enter your code then.
  • Press Enter or Send.
Getting a code:
  • If other user send code, you can download it.
  • Click in Download.
